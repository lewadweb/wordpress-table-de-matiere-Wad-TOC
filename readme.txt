Affiche une table des matières automatique et responsive via le shortcode [table_matiere].

== Description ==

Ce plugin ajoute automatiquement une table des matières responsive dans vos articles WordPress grâce au shortcode [table_matiere].  
Il est personnalisable via une page d’options où vous pouvez choisir les niveaux de titres, les couleurs, la largeur maximale, etc.  
Le sommaire s’affiche de façon élégante et s’adapte aux mobiles.

== Installation ==

1. Téléchargez et décompressez l’archive dans le dossier `/wp-content/plugins/`.
2. Activez le plugin via le menu Extensions dans WordPress.
3. Insérez le shortcode `[table_matiere]` dans vos articles pour afficher la table des matières.
4. Configurez les options via le menu **Table de matière** dans le tableau de bord WordPress.

== Questions fréquentes ==

= Comment utiliser ce plugin ? =
Utilisez simplement le shortcode `[table_matiere]` dans vos articles.

= Puis-je choisir quels niveaux de titres inclure ? =
Oui, vous pouvez sélectionner les niveaux de titres (h1 à h6) dans la page d’options du plugin.

= Le sommaire est-il responsive ? =
Oui, il s’adapte parfaitement à toutes les tailles d’écran.
